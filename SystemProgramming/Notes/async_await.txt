async / await
Ключевые слова async / await используются для формализации оабот с
многозадачными приложениями.
Главная особенность - подчеркнуть тот факт, что за пуск метожа происходит в асинхронном режиме.

Синхронный и асинхронный режимы принято разделять, поэтому
а) вызов (await) асинхронных методов разрешено только в других асинхронных методах
б) методы без асинхронной части не рекомендуется объявлять как async

вариант 1 (не нить)
Window				Task
loader --------->	  { FileContentAsync
loading...			  |
					  |
Content      <--------}
dic = await ------------> { ParseIniAsync
						  |
						  |
dic  <------------        }
LogDic(dic) ---->	  { LogDicAsync
}					  |
					  |
					  }

Window				Task
task --------->	  { FileContentAsync
loading...		  |
}				  | return
				  } --------> { ParseIniAsync

							  | return
